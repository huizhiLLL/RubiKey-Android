# 更新日志

## 0.2.0 - 2026-07-15

### 新增

- 完成 Moyu32、GAN v2/v3/v4、QiYi QYSC/Tornado V4 三类智能魔方协议支持。
- 支持将 12 种标准转动映射为点击、四方向滑动或无动作。
- 支持 BLE 后台连接和通过无障碍服务分发全局手势。
- 增加 Release 签名、R8 压缩、自适应图标和版本信息展示。

### 稳定性

- 补齐蓝牙运行时权限撤销时的异常处理。
- 增加 Android 12+ 数据备份排除规则。
- Release 构建纳入 lint、单元测试、签名和校验值验证。

### 验收范围

- Android 15 真机。
- 默认显示器和竖屏场景。
- 单魔方连接、后台运行、快速连续输入和同签名覆盖升级。

### 已知限制

- Android 12-14 和不同厂商后台策略尚未建立完整兼容矩阵。
- 暂不支持多魔方、复杂宏、横屏、多窗口、云同步和图像识别。
